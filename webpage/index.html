Hello IFC.js

IFC.js allows any user to review BIM models in non-BIM software (for example, a web browser), on all types of devices (computers, tablets and phones) for free.

IFC VIEWER

This library is an extension of web-ifc-three, which is the official IFCLoader for THREE.js.

This doesn't only parse and generate the Three.js geometry of IFC models in JavaScript, but also provides multiple tools to easily build BIM tools,

such as 3d dimensions, clipping planes, 2D plan navigation and generation, etc.
